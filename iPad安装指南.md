## 📱 方法一：直接传输到iPad（推荐）

### 步骤1：准备文件

压缩包已生成：
```
/Users/zhengtengfei/Documents/web_ai_game/pokemon-pinyin.zip
```

### 步骤2：传输到iPad

#### 选项A：使用AirDrop（最快）

1. **在Mac上**：
   - 找到 `pokemon-pinyin.zip` 文件
   - 右键点击 → 共享 → AirDrop
   - 选择您的iPad

2. **在iPad上**：
   - 接收文件
   - 文件会保存到"文件"应用的"下载"文件夹

#### 选项B：使用iCloud Drive

1. **在Mac上**：
   - 将 `pokemon-pinyin.zip` 复制到 iCloud Drive
   - 等待同步完成

2. **在iPad上**：
   - 打开"文件"应用
   - 进入 iCloud Drive
   - 找到 `pokemon-pinyin.zip`

#### 选项C：使用隔空投送（Handoff）

1. 通过数据线连接iPad和Mac
2. 使用Finder传输文件

### 步骤3：解压文件

在iPad上：
1. 打开"文件"应用
2. 找到 `pokemon-pinyin.zip`
3. 点击文件
4. iOS会自动解压到同一文件夹

### 步骤4：打开应用

1. 在"文件"应用中，进入解压后的 `pokemon-pinyin` 文件夹
2. 找到 `index.html` 文件
3. 长按 `index.html`
4. 选择"共享" → "拷贝"
5. 打开 **Safari 浏览器**
6. 在地址栏粘贴（会显示文件路径）
7. 点击前往

或者：
1. 直接点击 `index.html`
2. 选择用 **Safari** 打开

### 步骤5：添加到主屏幕

1. 在Safari中打开应用后
2. 点击底部的 **分享按钮** 📤
3. 向下滚动，找到 **"添加到主屏幕"**
4. 输入名称（默认：宝可梦拼音大搜查）
5. 点击 **"添加"**

### 步骤6：完成！

- 主屏幕上会出现应用图标 🎮
- 点击图标即可使用
- **完全离线运行**，无需网络

## 💻 方法二：本地服务器（快速测试）

### 适用场景
- 快速测试
- 多个设备同时使用
- 临时体验

### 步骤：

#### 1. 在Mac上启动服务器

打开终端，运行：
```bash
cd /Users/zhengtengfei/Documents/web_ai_game/pokemon-pinyin
python3 -m http.server 8000
```

看到提示：
```
Serving HTTP on :: port 8000 (http://[::]:8000/) ...
```

#### 2. 获取Mac的IP地址

新开一个终端窗口，运行：
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

会显示类似：
```
inet 192.168.1.100 netmask 0xffffff00 broadcast 192.168.1.255
```

记住这个IP地址（如：192.168.1.100）

#### 3. 在iPad上访问

1. 确保iPad和Mac连接到**同一个Wi-Fi**
2. 打开iPad的 **Safari 浏览器**
3. 在地址栏输入：
   ```
   http://192.168.1.100:8000
   ```
   （替换成你的IP地址）
4. 点击前往

#### 4. 添加到主屏幕

同方法一的步骤5

#### 注意事项
- ⚠️ 电脑必须保持开机
- ⚠️ 服务器必须保持运行
- ⚠️ iPad和Mac必须在同一网络

## ☁️ 方法三：云端部署（最方便）

### 推荐平台：Vercel（免费、快速）

#### 步骤1：注册Vercel账号

1. 访问 https://vercel.com
2. 点击 "Sign Up"
3. 使用GitHub账号登录（推荐）

#### 步骤2：上传项目

##### 方式A：拖拽上传（最简单）

1. 访问 https://vercel.com/new
2. 将整个 `pokemon-pinyin` 文件夹拖到页面上
3. 等待上传完成（可能需要几分钟，因为有1025张图片）
4. 点击 "Deploy"
5. 等待部署完成
6. 获得一个网址，如：`https://pokemon-pinyin.vercel.app`

##### 方式B：通过GitHub（推荐，方便更新）

1. 在GitHub创建新仓库：
   - 访问 https://github.com/new
   - 仓库名：`pokemon-pinyin`
   - 设为 Public
   - 点击 "Create repository"

2. 在Mac终端上传代码：
   ```bash
   cd /Users/zhengtengfei/Documents/web_ai_game/pokemon-pinyin
   git init
   git add .
   git commit -m "Initial commit: Pokemon Pinyin Quest v2.2"
   git branch -M main
   git remote add origin https://github.com/你的用户名/pokemon-pinyin.git
   git push -u origin main
   ```

3. 在Vercel导入：
   - 访问 https://vercel.com/new
   - 选择 "Import Git Repository"
   - 选择你的 `pokemon-pinyin` 仓库
   - 点击 "Deploy"

#### 步骤3：在iPad上使用

1. 打开Safari浏览器
2. 访问你的Vercel网址（如：`https://pokemon-pinyin.vercel.app`）
3. 点击底部的分享按钮 📤
4. 选择"添加到主屏幕"
5. 完成！

- ✅ 自动HTTPS加密

## 🆘 常见问题

### Q1: 文件太大，AirDrop失败？
**A**: 使用iCloud Drive或数据线传输

### Q2: Safari打开后是空白？
**A**:
1. 检查是否用Safari打开（不能用Chrome）
2. 检查文件路径是否正确
3. 尝试刷新页面

### Q3: 图片不显示？
**A**:
1. 确保 `images/pokemon/` 文件夹完整
2. 检查是否有网络连接（方法三首次需要）
3. 清除Safari缓存后重试

### Q4: 添加到主屏幕后找不到？
**A**:
1. 检查主屏幕的其他页面
2. 在主屏幕向下滑动，搜索"宝可梦"
3. 重新添加

### Q5: 语音不能播放？
**A**:
1. 检查iPad音量
2. 检查静音开关
3. 在设置中给Safari授予麦克风权限

### Q6: 进度丢失了？
**A**:
- 进度保存在Safari的本地存储中
- 不要清除Safari的网站数据
- 建议不要在"无痕浏览"模式下使用
